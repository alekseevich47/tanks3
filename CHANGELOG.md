## [1.0.1] - 2026-03-23

### 🎉 Стабильная версия (Исправления)

#### Исправлено
- **profile.js** — Убраны все пробелы в `getElementById()` ("tab-stats " → "tab-stats")
- **ui.js** — Исправлены пробелы в селекторах и сравнениях строк ("battle " → "battle")
- **state.js** — Убраны пробелы в названиях рангов ("Рядовой " → "Рядовой")
- **game.js** — Исправлены критические опечатки:
  - `fireBull et` → `fireBullet`
  - `pla yer` → `player`
  - `sk ills` → `skills`
  - `batt leStats` → `battleStats`
  - `hitOb stacle` → `hitObstacle`
  - `el se` → `else`
  - `ct x` → `ctx`
  - `TILE0.7` → `TILE * 0.7`
- **maps.js** — Очищены строки карт от лишних пробелов
- **style.css** — Исправлен `.profile-tabs` (margin-left → gap)

#### Удалено

- Функции `openAdminPanel()`, `closeAdminPanel()`, `adminSaveEconomy()`, `adminSaveParams()`
- Кнопка "НАСТРОЙКИ (АДМИН)" в правой панели
- Модальное окно `#admin-panel-overlay`
- Стили админ-панели из CSS

#### Восстановлено
- ✅ Отображение статистики в "Личном деле"
- ✅ Запуск боя при входе (кнопка "В ПЕРЕД!")

## [1.0.0] - 2026-03-22

### 🎉 Первый релиз (Alpha)

#### Добавлено
- Авторизация игроков через localStorage
- Игровой цикл: движение, стрельба, коллизии
- Режим боя "Зачистка" с 3 ботами
- 4 карты для режима "Зачистка" (случайный выбор)
- Система прогресса: ранги, опыт, монеты, отвага, топливо
- Экономика: снаряды (6 типов), ремкомплекты (4 типа)
- Интерфейс: ангар, бой, результат боя, профиль
- Раздел "Личное дело": Статистика, Рейтинг, Достижения
- Спец-умения: Щит, Ускорение стрельбы
- Логирование действий (logger.js)

#### Известные проблемы (исправлены в 1.0.1)
- Пробелы в getElementById() ломали работу интерфейса
- Опечатки в game.js предотвращали запуск боя
- Вёрстка "Личного дела" отображалась некорректно

## [1.2.0] - ЗАПЛАНИРОВАНО

### 🤖 Система ботов

#### Планируется
- Разделение ботов на типы: Простой, Турель, Сложный
- Настройка параметров каждого типа
- Улучшенный ИИ для "Сложного" бота
- Визуальное отличие типов ботов

### 🗺️ Карты и редактор

#### Планируется
- Вынос карт в отдельные JSON-файлы (`/maps/`)
- Простой редактор карт для создания/редактирования
- 3 новые карты для режима "Зачистка-ПРО"

### 🎮 Режим "Зачистка-ПРО"

#### Планируется
- Отдельная кнопка выбора режима
- 5 "Сложных" ботов + 1-2 "Турели" на карте
- Уникальные карты с разнообразными препятствиями

## [2.0.0] - БУДУЩЕЕ (Production)

### 🌐 Мультиплеер

#### Планируется
- Node.js сервер (Express + Socket.io)
- База данных (PostgreSQL/MySQL)
- Онлайн-бои (1v1, 2v2, 3v3)
- Синхронизация через WebSocket
- Серверная валидация (античит)

### 🔐 Безопасность

#### Планируется
- Отдельная админ-панель (не в клиенте)
- HTTPS, защита от XSS, CSRF
- Хеширование паролей
- Сессионная авторизация

### 🎨 Контент

#### Планируется
- Спрайтовая графика
- Звуковые эффекты
- Анимации
- Система кланов
- Достижения и ежедневные задания
- Рейтинг (лиги, сезонные награды)
